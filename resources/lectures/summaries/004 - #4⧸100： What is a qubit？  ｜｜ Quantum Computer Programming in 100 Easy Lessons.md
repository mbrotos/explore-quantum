# What is a qubit？

- The lecture contrasts bits and qubits by starting from familiar physical objects that can store two classical states, like coins, switches, and magnetic regions.
- A qubit has two basic states, `|0>` and `|1>`, but unlike a bit it can also be in a superposition `x|0> + y|1>` where `x^2 + y^2 = 1`.
- The same quantum rule generalizes to any system with multiple basic states: a valid state is a weighted combination whose squared amplitudes sum to 1.
- In principle, any physical system with two basic states could realize a qubit, though the course will stick to real amplitudes and defer complex numbers.
- Key takeaway: a qubit is best understood as a mathematically richer version of a two-state system, where amplitudes describe how much of each basic state is present.
