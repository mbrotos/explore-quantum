# Amplitude trees

- Extends ordinary probability trees into quantum analysis by replacing probabilities with amplitudes, including negative values.
- Walks through a two-qubit program step by step, showing how deterministic gates, Hadamards, and controlled toggles create branching paths.
- Explains how to compute final outcome amplitudes by multiplying along each path and then adding matching end states together.
- Highlights quantum interference: some paths cancel exactly, so states like `01` and `10` end with zero amplitude.
- Key takeaway: quantum programs are analyzed like probability trees, but signs matter because amplitudes can cancel and reshape the final state.
